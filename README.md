# How To Create a Responsive Login & Registration Form in HTML & CSS | Login and Registration Form

Um projeto simples de **formulário de login e cadastro responsivo** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **sistema de login e registro moderno**, com layout responsivo e design limpo.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Formulário de login
- Formulário de cadastro (registration)
- Layout responsivo
- Design simples e moderno

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
